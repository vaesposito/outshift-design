Strapi Components Reference

All the building blocks available in Strapi for composing pages. Use shared components to assemble content types, and content types to build pages.

Shared Components Single-Type Pages Collection Types

Shared Components

Reusable building blocks that can be embedded into any content type. These are the atomic pieces you compose pages from.

HeroBlock

Page hero section with title, description, and optional image.

FieldType
titlestringrequired
descriptiontextrequired
imagemedia
Used by: ResearchPage, FoundationalPrinciplesPage, SecurityPrivacyPage, AgentImpactMapPage, HaxPage, CognitiveFrameworksPage, SocietalImpactPage

CtaBlock

Call-to-action block with title, description, button, and image. Used as the "Build with SDK" section.

FieldType
titlestringrequired
descriptiontextrequired
buttonLabelstringrequired
buttonUrlstringrequired
imagemedia
Used by: ResearchPage, FoundationalPrinciplesPage, SecurityPrivacyPage, AgentImpactMapPage, HaxPage, CognitiveFrameworksPage, SocietalImpactPage (as sdk)

SectionHeader

Section header with title and optional subtitle for introducing page sections.

FieldType
titlestringrequired
subtitlestring
Used by: ResearchPage

SimpleCard

Basic card with title and description. Used for "Building Secure Systems" cards.

FieldType
titlestringrequired
descriptiontextrequired
Used by: SecurityPrivacyPage

PipelineStep

Numbered pipeline step with title, subtitle, description, quote, bullets, and deliverables.

FieldType
stepNumberstringrequired
titlestringrequired
subtitlestring
descriptiontext
quotetext
bulletstext
deliverablestext
Used by: FoundationalPrinciplesPage

CaseStudyCard

Case study card with title, description, problem statement, tags, and principles.

FieldType
titlestringrequired
descriptiontextrequired
problemtext
tagsTag[]
principlestext
Used by: FoundationalPrinciplesPage

ResearchApproachItem

Research approach item with title, description, explore bullets, and image. Used for alternating image/text rows.

FieldType
titlestringrequired
descriptiontextrequired
exploreBulletstextrequired
imagemedia
imageAltstring
Used by: SecurityPrivacyPage

UseCaseEntry

Use case card with title, description, and tags.

FieldType
titlestringrequired
descriptiontextrequired
tagsTag[]
Used by: SecurityPrivacyPage

PatternPanel

Pattern panel with key, title, description, and multiple example images. Used for HAX interaction patterns.

FieldType
keystringrequired
titlestringrequired
descriptiontextrequired
exampleImagesmedia[]
Used by: HaxPage

ResponsibleStep

Framework step with label, title, image, bullets, and optional template reference link.

FieldType
labelstringrequired
titlestringrequired
imagemedia
imageAltstring
bulletstextrequired
templateLabelstring
templateLinkstring
Used by: SocietalImpactPage

MediaBlock

Flexible media block supporting images and videos with alt text and captions.

FieldType
mediaTypeenumrequired
imagemedia
videoUrlstring
altTextstringrequired
captionstring
Used by: ResearchCard, Initiative

Tag

Reusable label tag for categorization. Colors: cyan, violet, green, orange, default.

FieldType
labelstringrequired
urlstring
colorenum
Used by: CaseStudyCard, UseCaseEntry, BlogPost, ResearchCard

NavItem

Navigation menu item with label, URL, dropdown flag, and sort order.

FieldType
labelstringrequired
hrefstringrequired
hasDropdownboolean
orderinteger
Used by: Homepage

ArrowLink

Call-to-action arrow link with label, URL, and external flag.

FieldType
labelstringrequired
urlstringrequired
isExternalboolean
Used by: Homepage, ResearchCard, HaxPage

SocialLink

Social media link with platform icon. Platforms: x, linkedin, youtube, github, dribbble.

FieldType
platformenumrequired
urlstringrequired
ariaLabelstringrequired
Used by: Homepage

SeoMeta

SEO metadata for pages: title (max 60 chars), description (max 160 chars), OG image, canonical URL.

FieldType
metaTitlestringrequired
metaDescriptiontext
ogImagemedia
canonicalUrlstring
Used by: Homepage, BlogPost, ResearchCard, Initiative, ResearchPage, FoundationalPrinciplesPage, SecurityPrivacyPage, AgentImpactMapPage, HaxPage, CognitiveFrameworksPage, SocietalImpactPage

Single-Type Pages

Each single type represents one page. Edit it directly in Strapi — no need to create entries.

Homepage Single Type

Homepage configuration: hero content, section titles, navigation, footer links, and social links.

heroTitle string heroSubtitle string heroDescription text heroCtaLabel string heroCtaUrl string initiativesSectionTitle string initiativesSectionSubtitle text researchSectionTitle string researchSectionDescription text blogSectionTitle string blogSectionDescription text navigation NavItem[] footerLinks ArrowLink[] socialLinks SocialLink[] seo SeoMeta

ResearchPage Single Type

Research listing page with hero, section header, and CTA. Research cards are managed separately as a collection.

hero HeroBlock sectionHeader SectionHeader cta CtaBlock seo SeoMeta

HaxPage Single Type

The Human-Agent Experience page: hero with optional video, interaction patterns, research section, and SDK CTA.

hero HeroBlock heroVideo string patternsTitle string patternsDescription text patterns PatternPanel[] researchTitle string researchDescription text researchLink ArrowLink researchImage media sdk CtaBlock seo SeoMeta

FoundationalPrinciplesPage Single Type

Foundational Principles research page: hero, pipeline steps, case studies, and SDK CTA.

hero HeroBlock pipelineTitle string pipelineSubtitle text steps PipelineStep[] caseStudiesTitle string caseStudiesDescription text caseStudies CaseStudyCard[] sdk CtaBlock seo SeoMeta

CognitiveFrameworksPage Single Type

Cognitive Frameworks research page: hero, theoretical section, rich text body, banner items, and SDK CTA.

hero HeroBlock theoreticalTitle string theoreticalDescription text body richtext bannerItems text sdk CtaBlock seo SeoMeta

SocietalImpactPage Single Type

Societal Impact research page: hero with list items, responsible framework steps, and SDK CTA.

hero HeroBlock heroListItems text frameworkTitle string frameworkDescription text steps ResponsibleStep[] sdk CtaBlock seo SeoMeta

SecurityPrivacyPage Single Type

Security & Privacy research page: hero, secure systems cards, key questions, research approach items, use cases, and SDK CTA.

hero HeroBlock secureSystemsTitle string secureCards SimpleCard[] keyQuestionsTitle string keyQuestions text researchApproachTitle string researchApproachDescription text researchItems ResearchApproachItem[] useCases UseCaseEntry[] sdk CtaBlock seo SeoMeta

AgentImpactMapPage Single Type

Agent Impact Map research page: hero, template instructions and methodology (richtext), diagram image, and SDK CTA.

hero HeroBlock templateTitle string templateSubtitle text instructions richtext methodology richtext diagram media sdk CtaBlock seo SeoMeta

Collection Types

Repeatable entries managed as lists. Create as many as needed — they appear on listing pages and can be referenced elsewhere.

BlogPost Collection

Blog articles shown on the homepage and blog listing. Supports rich text content, cover images, and tags.

title string description text slug uid author string publishDate date readTime string content richtext coverImage media tags Tag[] seo SeoMeta

ResearchCard Collection

Research cards displayed on the research listing page. Categories: research-area or design-tool.

title string description text slug uid category enum media MediaBlock tags Tag[] link ArrowLink order integer seo SeoMeta

Initiative Collection

Featured initiative cards shown on the homepage. Supports reversed layout and sort order.

title string description text slug uid badge string media MediaBlock link ArrowLink reversed boolean order integer seo SeoMeta
