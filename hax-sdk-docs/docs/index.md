# HAX SDK

The HAX SDK gives developers everything they need to integrate agents into their apps — without losing clarity, structure, or control. Use structured schemas, prebuilt components, and clear boundaries to keep agent behavior collaborative and predictable.

## Getting Started

- [Installation](installation.md) — Set up HAX in your project
- [Custom Repositories](custom-repositories.md) — Extend HAX with custom component registries

## Components

HAX provides a rich set of components that integrate with AI agents through CopilotKit:

| Component | Description |
|-----------|-------------|
| [Source Attribution](components/source-attribution.md) | Transparent citations with verifiable sources |
| [Timeline](components/timeline.md) | Chronological activity tracking with status indicators |
| [Details Dashboard](components/details-dashboard.md) | Comprehensive data display for entities and processes |
| [Data Visualizer](components/data-visualizer.md) | Charts and graphs built on Chart.js |
| [Mindmap](components/mindmap.md) | Interactive hierarchical relationship visualization |
| [Form Builder](components/form-builder.md) | Dynamic forms with validation and multiple field types |
| [Code Editor](components/code-editor.md) | Syntax highlighting with multi-language support |
| [Inline Rationale](components/inline-rationale.md) | AI-driven assessments with intent-based theming |
| [Capability Manifest](components/capability-manifest.md) | Agent capability and status display |
| [Workshop Card](components/workshop-card.md) | Event display with attendees and RSVP |
| [Contextual Explanation](components/contextual-explanation.md) | Structured explanations for agent decisions |
| [Findings](components/findings.md) | Key insights with source attribution |
| [Diagnostic Report](components/diagnostic-report.md) | Diagnostic findings with confidence levels |
| [Rich Text Editor](components/rich-text-editor.md) | Rich text editing with collaborative features |
| [Co-Editing Interface](components/co-editing-interface.md) | Shared revision and comment workspace |
| [Thinking Process](components/thinking-process.md) | AI reasoning and workflow visualization |
