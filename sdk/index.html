    HAX SDK

    The HAX SDK gives developers everything they need to integrate agents into their apps — without losing clarity, structure, or control. Use structured schemas, prebuilt components, and clear boundaries to keep agent behavior collaborative and predictable.

    Getting Started

    • Installation — Set up HAX in your project
    • Custom Repositories — Extend HAX with custom component registries

    Components

    HAX provides a rich set of components that integrate with AI agents through CopilotKit:

    Component Description
    Source Attribution Transparent citations with verifiable sources
    Timeline Chronological activity tracking with status indicators
    Details Dashboard Comprehensive data display for entities and processes
    Data Visualizer Charts and graphs built on Chart.js
    Mindmap Interactive hierarchical relationship visualization
    Form Builder Dynamic forms with validation and multiple field types
    Code Editor Syntax highlighting with multi-language support
    Inline Rationale AI-driven assessments with intent-based theming
    Capability Manifest Agent capability and status display
    Workshop Card Event display with attendees and RSVP
    Contextual Explanation Structured explanations for agent decisions
    Findings Key insights with source attribution
    Diagnostic Report Diagnostic findings with confidence levels
    Rich Text Editor Rich text editing with collaborative features
    Co-Editing Interface Shared revision and comment workspace
    Thinking Process AI reasoning and workflow visualization

    Add On Description
    Chat Commands Command-driven chat with @, +, / syntax
    File Upload Seamless file sharing with AI agents
